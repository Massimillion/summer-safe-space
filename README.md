## Project info

# SquirrelBox Storage

A full-stack web platform built for a college student summer storage business. 
The goal was to consolidate everything — the website, CRM, bookings, and 
scheduling — into one platform instead of juggling multiple tools like Zapier, 
Airtable, and Zoho.

## What it does
- Student-facing website and booking flow
- Scheduling system for pickups and drop-offs
- CRM to manage student accounts and storage details
- Payment processing via Stripe

## Why I built it
Running a seasonal storage business meant dealing with too many disconnected 
tools. I wanted one platform I controlled, with Stripe as the only third-party 
dependency for payments.

## Tech & Tools
- Built with Lovable
- Stripe API for payments
- Not a demo, used daily
